Mise à Jour du Gestionnaire d’Invitation de V1.0.0 à V1.1.0 – Activation du Suivi Contextuel

Étape 1: Télécharger les fichiers d’installation

Cliquez ce lien pour télécharger l'outil du gouvernement du Canada du Gestionnaire d'Invitation: Télécharger le fichier Zip du Gestionnaire d'Invitation
Cliquez pour ouvrir le fichier .zip ou utilisez n’importe quel outil pour décompresser le fichier et enregistrez son contenu sur votre lecteur local. Dans le fichier ZIP vous trouverez ce qui suit:

  • Dossier “invitation-manager”:
    • InvitationManager.js
    • Overlay.js
    • Overlay.css
    • config.json
    • im.json

Étape 2: Télécharger les fichiers sur votre serveur

Ceux qui suit sont Uniquement les fichiers qui nécessitent une mise à jour – copiez les dans la l’emplacement des anciens fichiers:

  1. config.json
  2. InvitationManager.js

Étape 3: Configurer les paramètres – Mettre à jour les paramètres

  1. dbURL

    Ouvrer le fichier “config.json” dans le dossier “invitation-manager” ou dans votre dossier enregistré alternatif et localiser le paramètre dbURL.
    Copier et coller la valeur de l’ancien fichier “config.json”.

    Example: “/invitation-manager/im.json”.
       dbURL: "/invitation-manager/im.json"

  2. Adobe

    Si Adobe Analytics est mis en œuvre sur votre site, vous pouvez activer le suivi des invitations contextuelles du gestionnaire d’invitation dans le fichier “config.json”. Le suivi Adobe Analytics pour le gestionnaire d’invitation est actuellement disponible sur la version 3, la version 4 et la version Run du code Non-Canada.ca. Si Adobe Analytics n’est pas mis en œuvre sur votre site vous ne bénéficierez pas de cette fonctionnalité.
    Ouvrer le fichier “config.json” dans le dossier “invitation-manager” ou dans votre dossier enregistré alternatif et localiser le paramètre Adobe, la valeur doit être égale “yes” ou “no”.

    • Entrer "Adobe": "no", si le suivi Adobe Analytics de votre site du département n’est pas mis en œuvre ou vous préfériez ne pas activer le suivi Adobe Analytics.
    • Entrer "Adobe": "yes", si le suivi Adobe Analytics de votre site du département est mis en œuvre et vous préfériez activer le suivi Adobe Analytics.
