Mise à Jour du Gestionnaire d’Invitation de V1.0.0 à V1.1.0 – Correction de l'icône de fermeture [X] pour les invitations contextuelles avec des longs titres

Étape 1: Télécharger les fichiers d’installation

Cliquez ce lien pour télécharger l'outil de Gestionnaire d’Invitation du gouvernement du Canada: Télécharger le fichier Zip du Gestionnaire d'Invitation
Cliquez pour ouvrir le fichier .zip ou utilisez n’importe quel outil pour décompresser le fichier et enregistrez son contenu sur votre lecteur local. Dans le fichier ZIP vous trouverez ce qui suit:

  • Dossier “invitation-manager”:
    • InvitationManager.js
    • Overlay.js
    • Overlay.css
    • config.json
    • im.json

Étape 2: Télécharger les fichiers sur votre serveur

Les fichiers suivants sont les SEULS qui nécessitent une mise à jour – copiez-les et remplacez-les à l’emplacement des anciens fichiers:

  1. InvitationManager.js
  2. Overlay.css
  3. config.json

Étape 3: Configurer les paramètres – Mettre à jour les paramètres

  1. dbURL

    Ouvrer le fichier “config.json” dans le dossier “invitation-manager” ou dans votre dossier enregistré alternatif et localiser le paramètre "dbURL".
    La valeur de ce paramètre doit indiquer l'emplacement du fichier im.json. Mettez-la à jour pour qu'elle corresponde à la valeur que vous aviez dans le fichier config.json précédent. Si l'emplacement du fichier im.json a changé, mettez à jour la valeur pour refléter le nouvel emplacement.

    • Example: “/invitation-manager/im.json”.

  2. Adobe

    Si Adobe Analytics est déjà mis en œuvre sur votre site, vous pouvez activer le suivi des invitations contextuelles du gestionnaire d’invitation dans le fichier “config.json”. Le suivi Adobe Analytics pour le gestionnaire d’invitation est actuellement disponible sur la version 3, la version 4 et la version exécutable du code Non-Canada.ca. Si Adobe Analytics n’est pas mis en œuvre sur votre site vous ne bénéficierez pas de cette fonctionnalité.
    Ouvrer le fichier “config.json” dans le dossier “invitation-manager” ou dans votre dossier enregistré alternatif et localiser le paramètre "Adobe", la valeur doit être égale “yes” ou “no”.

    • Entrer "Adobe": "no", si le site de votre département n'a pas mis en place le suivi Adobe Analytics ou si vous préférez ne pas activer le suivi Adobe Analytics.
    • Entrer "Adobe": "yes", si le site de votre département a mis en place le suivi d’Adobe Analytics et que vous préfériez activer le suivi Adobe Analytics.
